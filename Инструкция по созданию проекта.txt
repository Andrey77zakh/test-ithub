Зайти на сайт GitHub и авторизоваться под своей учётной записью. Нажать зелёную кнопку «New» в разделе репозиториев. Ввести понятное название проекта — оно будет отображаться в URL и интерфейсе. При необходимости добавить краткое описание. Поставить галочку при желании на «Add a README file», чтобы репозиторий сразу содержал базовый файл с информацией. Выбрать тип видимости - обычно «Public», если проект не приватный. Остальные настройки (лицензия, .gitignore и т.д.) можно оставить по умолчанию или добавить позже. Нажать «Create repository» — и проект будет готов: в нём можно сразу просматривать файлы, клонировать репозиторий, добавлять код и вести разработку.
